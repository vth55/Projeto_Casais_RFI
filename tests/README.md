# Testes

Esta pasta contém os testes do projeto.

## Estrutura

```
tests/
├── unit/          # Testes unitários
├── integration/   # Testes de integração
└── e2e/          # Testes end-to-end
```

## Executar Testes

```bash
# Testes unitários
npm test

# Testes com coverage
npm test -- --coverage
```
