CASAIS RFI - PASTA DE FEEDBACK VISUAL PARA O CLAUDE

O utilizador deve colocar aqui os prints (imagens .png, .jpg) da interface, tanto erros como ideias visuais.

COMO USAR ESTA PASTA:
1. Tira o print do ecrã (ex: "erro_mobile.png").
2. Grava-o nesta pasta (`_prints/erro_mobile.png`).
3. No terminal, escreve ao Claude: 
   "Claude, vai ler a imagem na pasta _prints/erro_mobile.png e corrige o espaçamento CSS disto no Frontend_App."

O Claude vai ler nativamente o ficheiro aqui deixado e escrever o código Tailwind ou UI necessário de forma muito cirúrgica! Podes ir adicionando ou apagando as fotos conforme a tua gestão.
