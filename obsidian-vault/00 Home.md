# 00 Home

## What This Project Is
- Industrial PWA for Casais fleet operations and equipment usage tracking.
- Core responsibilities: RFID session tracking, costs and CO2, maintenance, breakdown reporting, Procore integration.
- Main stack: React 19 + Vite + Tailwind + Zustand + Recharts; Firebase Functions v2 + Firestore + Storage.
