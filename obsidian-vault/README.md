# Obsidian Vault

Curated, AI-friendly memory vault for `Projeto_Casais_RFI`.

Purpose:
- give Claude, Codex, Gemini, and future agents one stable memory layer
- preserve durable context without forcing long chats
- keep operational knowledge visible and editable by humans

How to use:
- open this folder as an Obsidian vault
- start at [[00 Home]]
- for AI agents, prefer reading:
  - [[00 Home]]
  - [[01 AI Working Guide]]
  - [[02 Memory Strategy]]
  - [[03 Planning Workflow]]
  - the domain/system note relevant to the task

Principles:
- short notes, high signal
- each note links to sources and code paths
- durable facts live here; ephemeral session chatter does not
- project docs remain the source of detail; this vault is the source of orientation
- vault maintenance rules live in [[33 Vault Maintenance Rules]]
