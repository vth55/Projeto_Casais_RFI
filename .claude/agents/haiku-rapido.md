# Agente Haiku - Tarefas Rápidas

Usa este agente para:
- Pesquisas no código
- Análises rápidas de ficheiros
- Formatação de texto
- Verificações simples
- Tarefas que não precisam de modificar ficheiros

Este agente é READ-ONLY e usa o modelo Haiku 4 que é mais económico.
