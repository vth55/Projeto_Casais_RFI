**Foco:** Debug e fix do erro "Subprocess initialization did not complete within 60000ms" na extensão Claude Code do Antigravity (VS Code fork)
**Branch:** migration/claude-setup

## O que foi feito
- Diagnosticado o erro de timeout na extensão Claude Code do Antigravity
- Descartada hipótese de falta de autenticação (credenciais WSL estavam OK)
- Descartada hipótese de falta do `claude.exe` no Windows PATH (extensão usa binário embutido)
- Identificado o servidor MCP `mcp-search` do plugin `claude-mem@thedotmack` como causa raiz
- Lidos os logs da extensão em `AppData/Roaming/Antigravity/logs/` — erro explícito: `Missing environment variables: _R%/`
- Identificado que a versão 12.7.5 do claude-mem usa sintaxe bash `${_R%/}` que o Windows interpreta como variável de ambiente inválida `_R%/`
- Fix aplicado: alterado `enabledPlugins.claude-mem@thedotmack` de `true` para `false` em `C:\Users\vitor\.claude\settings.json`
