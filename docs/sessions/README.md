# Sessions

Esta pasta guarda wrap-ups e evidencias de sessao.

- Serve para auditoria e historico de execucao.
- Nao substitui `docs/ROADMAP_EXECUCAO.md`, `FINDINGS.md` ou a documentacao canonica.
- Screenshots e dumps temporarios de Playwright nao devem entrar no indice principal do projeto.

## Estrutura

- `YYYY-MM-DD_HH-MM.md` -> wrap-ups de sessao
- `evidence/YYYY-MM-DD/` -> screenshots e evidencias preservadas
- `docs/archive/sessions/` -> dumps temporarios e artefactos que nao devem ficar no espaco ativo

## Regra

Se uma screenshot nao sustenta um bug, validacao, demo ou regressao concreta, nao deve viver aqui.
