## 🎯 KPIs ESSENCIAIS DO SETOR

### **1. OEE (Overall Equipment Effectiveness)** ⭐⭐⭐
**Eficiência Global dos Equipamentos**

**Fórmula:**
```
OEE = Disponibilidade × Desempenho × Qualidade
```

**Componentes:**
- **Disponibilidade:** Tempo disponível / Tempo total
- **Desempenho:** Velocidade real / Velocidade ideal
- **Qualidade:** Produção conforme / Produção total

**Meta Típica:** 85% (World Class)

**Porquê é Importante:**
- Indica eficácia geral do equipamento
- Identifica oportunidades de melhoria
- Padrão internacional da indústria

**Visualização:**
- Gauge (medidor) circular
- Gráfico de barras (comparação)
- Breakdown por componente

---

### **2. MTBF (Mean Time Between Failures)** ⭐⭐⭐
**Tempo Médio Entre Falhas**

**Fórmula:**
```
MTBF = Total de Horas Operacionais / Número de Falhas
```

**Unidade:** Horas

**Porquê é Importante:**
- Indica confiabilidade do equipamento
- Ajuda a prever quando vai falhar
- Base para planeamento de manutenção

**Visualização:**
- Gráfico de linha (evolução temporal)
- Gráfico de barras (comparação entre máquinas)
- Comparação com meta

---

### **3. MTTR (Mean Time To Repair)** ⭐⭐⭐
**Tempo Médio para Reparo**

**Fórmula:**
```
MTTR = Total de Tempo de Reparo / Número de Reparos
```

**Unidade:** Horas

**Porquê é Importante:**
- Indica eficiência da manutenção
- Ajuda a otimizar processos de reparo
- Reduz downtime

**Visualização:**
- Gráfico de linha (evolução temporal)
- Gráfico de barras (comparação)
- Comparação com meta

---

### **4. Taxa de Disponibilidade** ⭐⭐⭐
**Availability Rate**

**Fórmula:**
```
Taxa de Disponibilidade = (Tempo Disponível / Tempo Total) × 100
```

**Unidade:** Percentagem (%)

**Porquê é Importante:**
- Indica % de tempo que máquina está operacional
- KPI fundamental para produtividade
- Meta típica: 90-95%

**Visualização:**
- Gráfico de barras
- Gráfico de linha (evolução)
- Gauge (medidor)
- Breakdown por motivo de indisponibilidade

---

### **5. CPMV (Custo de Manutenção sobre Valor de Reposição)** ⭐⭐
**Maintenance Cost over Replacement Value**

**Fórmula:**
```
CPMV = (Custo Total de Manutenção / Valor de Reposição) × 100
```

**Unidade:** Percentagem (%)

**Porquê é Importante:**
- Indica se vale a pena manter ou substituir
- Regra: Se CPMV > 50%, considerar substituição
- Ajuda em decisões de investimento

**Visualização:**
- Gráfico de barras
- Comparação entre máquinas
- Alertas quando > 50%

---

### **6. Tempo de Inatividade (Downtime)** ⭐⭐
**Tempo que máquina está parada**

**Fórmula:**
```
Downtime = Tempo Total - Tempo Operacional
```

**Unidade:** Horas

**Porquê é Importante:**
- Indica perdas de produtividade
- Ajuda a identificar problemas
- Base para melhorias

**Visualização:**
- Gráfico de barras (por motivo)
- Gráfico de área (acumulado)
- Comparação entre máquinas

---

### **7. Taxa de Utilização (Uptime)** ⭐⭐
**% de tempo que máquina está a trabalhar**

**Fórmula:**
```
Taxa de Utilização = (Horas Trabalhadas / Horas Disponíveis) × 100
```

**Unidade:** Percentagem (%)

**Porquê é Importante:**
- Indica eficiência de uso
- Diferente de Disponibilidade (pode estar disponível mas não usada)
- Meta típica: 70-80%

**Visualização:**
- Gráfico de barras
- Gráfico de linha (evolução)
- Comparação entre máquinas
